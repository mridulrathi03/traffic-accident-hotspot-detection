# 🚦 Traffic Accident Hotspot Detection

## 📌 Overview
This project identifies accident-prone areas using data analysis and clustering techniques. It helps in detecting high-risk zones for better road safety planning.

## 🎯 Objective
To analyze accident data and highlight hotspot locations where accidents occur frequently.

## ⚙️ Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn (K-Means Clustering)
- Streamlit (for visualization)

## 📊 Features
- Data preprocessing and cleaning
- Clustering using K-Means algorithm
- Identification of accident hotspots
- Visualization of high-risk areas

## 📁 Dataset
The dataset contains accident records used to analyze and detect hotspot regions.
